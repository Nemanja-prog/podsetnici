## git clone
1. Kopiraj SSH link napravljenog repozitorijuma sa GitHub-a
2. Napravi i uđi u direktorijum u terminalu
3. Kucaj `git clone SSH link`

## git remote -v
1. Uđi u direktorijum napravljenog repozitorijuma
2. Kucaj `git remote -v`

## Dodavanje fajla u repozitorijum
1. Kucaj `touch ime fajla`
2. Kucaj `git add ime fajla`

## Provera Statusa
1. Kucaj `git status`, crveno nije postavljen, zeleno jeste

## Snimanje promena
1. Kucaj `git commit -m (poruka) ime fajla/ova`
2. Kucaj git status
3. `"nothing to commit, working tree clean"` promena snimljena

## Provera istorije
1. Kucaj `git log`

## Modifikovsnje fajla ili dva
1. Kucaj `code .` da otvoriš Visual Studio Code
2. Radi izmenu i sačuvaj (ako autosave nisi uključio)
3. Kucaj git add ime fajla, pa git status da proveriš stanje

## Kačenje na GitHub
1. Kucaj `git push`
